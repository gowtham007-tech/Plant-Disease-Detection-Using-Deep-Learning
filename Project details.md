PROJECT TITLE
Plant Disease Detection Using Deep Learning
PROJECT OVERVIEW
The Plant Disease Detection Using Deep Learning System is a web-based application that uses Artificial Intelligence and Deep Learning techniques to identify plant diseases from leaf images. The system helps farmers and agricultural experts detect diseases at an early stage and take preventive measures to improve crop health and yield.
PROBLEM STATEMENT
Plant diseases can significantly reduce crop productivity and quality. Traditional disease detection methods rely on manual inspection, which can be time-consuming and inaccurate. Therefore, a Deep Learning-based Plant Disease Detection System is required to identify plant diseases quickly and accurately using leaf images.
PROJECT OBJECTIVE
Main Objective:
To develop a Deep Learning-based system that detects plant diseases from leaf images and provides accurate disease predictions.
Specific Objectives:
Collect plant leaf images.
Process and analyze images using Deep Learning algorithms.
Identify plant diseases accurately.
Display disease prediction results instantly.
Assist farmers in early disease detection.
Improve crop management and productivity.
FEATURES

INPUT PARAMETERS
Plant Name
Leaf Image
Crop Type
Image Quality
Plant Age (Optional)
OUTPUT
Healthy Plant
If no disease symptoms are detected in the uploaded leaf image.
Diseased Plant
If disease symptoms are detected, the system displays the disease name and prediction result.
